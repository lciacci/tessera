Toggle per-response usage summary.

```
/usage-summary off    # Hide usage stats after each response
/usage-summary on     # Show usage stats after each response
```

When on, you see a compact summary after each Claude response:
> 📊 Route: DEEPSEEK_PRO (by qwen3) · est: ~$0.003 · today: 459 decisions

Shows: model used, classifier, estimated cost, and today's total decisions.
