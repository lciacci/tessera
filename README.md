# Tessera

Personal agentic coding framework for Claude Code, originally inspired by [Maggy](https://github.com/alinaqi/maggy). Tessera retains Maggy's foundational ideas (skills/rules/hooks architecture, TDD-via-Stop-hook, PreCompact recovery patterns) while diverging substantially in design: project profiles, structured event logging, suggestion-gate pattern for heavy machinery, multi-model abstraction via LiteLLM, and a healthcare-aware profile extension.

Status: in development. See `tess-design-principles.md` for the full design rationale.
